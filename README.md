A full-stack book store management application built with Node.js, Express.js, and MongoDB.

 🚀 Features

- Books & Authors CRUD operations
- User registration & login with JWT authentication
- Password reset via email (Nodemailer)
- File upload with Multer
- Input validation with Joi
- API security with Helmet & CORS
- Responsive frontend SPA (HTML/CSS/JS)
